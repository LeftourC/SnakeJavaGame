# Custom Assets for Snake Game

Put PNG files (32Ã—32) in this folder to replace the generated graphics:

- `food.png` - Food/apple sprite
- `snake_body.png` - Snake body segment (coming soon)
- `snake_head_up.png`, etc. (coming soon)
- `grid_cell.png` (coming soon)

The game loads these automatically when present. Rebuild the project (mvn compile) after adding new assets.

**Free asset sources:**
- [Kenney.nl](https://kenney.nl/assets) - CC0 game assets
- [itch.io](https://itch.io/game-assets/free) - Free game art
- [OpenGameArt.org](https://opengameart.org/) - Open source game art
